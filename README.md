## 🚀 Getting Started

### 📋 **Prerequisites**
- PHP 8.1+
- Laravel 9+
- MySQL 8.0+
- Node.js 18+
- Composer
- NPM/Yarn

### ⚡ **Quick Setup**
```bash
# Clone the repository
git clone https://github.com/your-org/qiass.git

# Install backend dependencies
composer install

# Install frontend dependencies
cd frontend/qiass-admin
npm install

# Setup environment
cp .env.example .env
php artisan key:generate

# Run migrations
php artisan migrate --seed

# Start development servers
php artisan serve  # Backend
npm run dev       # Frontend
```
